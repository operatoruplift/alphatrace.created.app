# AlphaTrace Examples

This directory contains practical examples demonstrating how to use AlphaTrace in various scenarios.

## Available Examples

### [Basic Setup](basic-setup/)
Learn the fundamentals of setting up AlphaTrace in your application.

### [Advanced Usage](advanced-usage/)
Explore advanced features and integration patterns.

## Running Examples

Each example directory contains its own README with specific instructions. Generally:

```bash
cd examples/[example-name]
npm install
npm start
```

## Contributing Examples

We encourage the community to contribute examples! If you have a use case or integration pattern that might help others, please submit a pull request.
