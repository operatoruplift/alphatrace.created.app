# Getting Started with AlphaTrace

## Installation

### Quick Setup

```bash
npm install @alphatrace/core
```

### Configuration

Create a configuration file:

```javascript
// alphatrace.config.js
export default {
  apiKey: 'your-api-key',
  endpoint: 'https://alphatrace.created.app/api',
  environment: 'production'
}
```
